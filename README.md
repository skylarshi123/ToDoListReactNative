## Setup Instructions

1. **Prerequisites**
   ```bash
   # Install Expo CLI globally
   npm install -g expo-cli
   ```

2. **Installation**
   ```bash
   # Clone the repository
   git clone [repository-url]

   # Install dependencies
   cd todo-app
   npm install
   ```

3. **Running the App**
   ```bash
   # Start the development server
   npx expo start
   ```
